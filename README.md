## Overview
Recode the ls command

## Project Structure
- `Makefile`: Automation tool
- `libft/`: Library used in the project (42)

## How to Use
1. Clone the repository.
2. Run the Makefile with ```make```
3. Use the ```ft_ls``` command, eg: ```./ft_ls -a```
